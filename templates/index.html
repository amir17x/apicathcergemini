Gmail & Gemini API Creator

How to use

This bot automatically creates Gmail accounts and generates Google Gemini API keys for you.

Start by messaging the bot on Telegram:

  • Use /start to get started
  • Use /create to create a new Gmail account and API key
  • Use /status to check the status of your requests
Setup Instructions
  1. Find the bot on Telegram at: @YourBotUsername
  2. Send the /start command to initialize the bot
  3. Use /create to begin account creation process
  4. Follow the prompts to specify proxy settings (if needed)
  5. Wait for the bot to create your account and API key
  6. Credentials will be sent to you when the process completes
Important Notes
  • Account creation process may take 3-5 minutes
  • You may need a proxy/VPN if in a region with restricted access
  • Phone verification may be required by Google (not automated)
  • Keep your credentials secure - do not share them!
